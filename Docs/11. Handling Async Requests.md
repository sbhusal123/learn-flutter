# Handling Asynchronous Code / Requests

Let's consider the snippet below:

```dart
void getData(){

  // simulate a delay
  Future.delayed(Duration(seconds: 2), () {
    print('Data fetched');
  });

  print('end of function');

}

void main(){
  getData();
}
```

Here, the output will be:

```sh
end of function
Data fetched
```

A problem we ran here is, `print('end of function');` gets executed before the `print('Data fetched');`. It should have waited for the data to be fetched.


## Workaround solution:

```dart
void getData() async {
  String username = await Future.delayed(Duration(seconds: 2), (){
    return 'Surya';
  });

  String bio = await Future.delayed(Duration(seconds: 1), (){
    return "Student";
  });

  print("$username - $bio");
}

void main() {
  getData();
}
```

**Basic signature** for defining a Async functions:

```dart
Future<ReturnType> funcName(params...) {
	
}
```

**Example**
```dart
Future<String> getData()  {
  return Future.delayed(Duration(seconds: 2), (){
    return 'Data';
  });
}


void main() async {
  String username = await getData();

  print(username);
}
```

## Example of getting data from API

```dart
import 'package:flutter/material.dart';

import 'package:http/http.dart';
import 'dart:convert';

class Loading extends StatefulWidget {
  const Loading({super.key});

  @override
  State<Loading> createState() => _LoadingState();
}

class _LoadingState extends State<Loading> {

  void getData() async {

    // making http request
    Response response = await get(Uri.parse('https://jsonplaceholder.typicode.com/todos/1'));

    // decoding JSON string into Dart object ( Map or List )
    Map data = jsonDecode(response.body);

    print(data);

    print(data['title']);
  }

  @override
  void initState() {
    super.initState();
    getData();
  }


  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Text('Loading screen')
    );
  }
}

```
