# Handling List Of States:

Suppose we have a list of items as below in the state:

```dart
  List<String> quotes = [
    'Be yourself; everyone else is already taken.',
    'I have not failed. I\'ve just found 10,000 ways that won\'t work.',
    'The only way to do great work is to love what you do.',
  ];
```

We have a state defined as below:

```dart
class QuoteList extends StatefulWidget {
  const QuoteList({super.key});

  @override
  State<QuoteList> createState() => _QuoteList();
}
```

And our widget is:

```dart
class _QuoteList extends State<QuoteList> {

  List<String> quotes = [
    'Be yourself; everyone else is already taken.',
    'I have not failed. I\'ve just found 10,000 ways that won\'t work.',
    'The only way to do great work is to love what you do.',
  ];

  
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      backgroundColor: Colors.grey[200],
      appBar: AppBar(
        title: Text("Awesome Quotes"),
        backgroundColor: Colors.redAccent,
        centerTitle: true,
      ),
      body: Column(
        children: quotes.map((quote) {
            return Text(quote);
        }).toList()
      )
    );
  }
}
```

We can cycle through a list of states by using a custom class that has members which we want to display in a state. Let's define a class as below:

```dart
class Quote {
  final String text;
  final String author;
  Quote({this.text, this.author});
}
```

Now we can use this class to define our state as below:

```dart
class _QuoteList extends State<QuoteList> {

  List<Quote> quotes = [
    Quote(text: "Stay Hungry, Stay Foolish", author: "Steve Jobs"),
    Quote(text: "The only way to do great work is to love what you do", author: "Steve Jobs"),
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      backgroundColor: Colors.grey[200],
      appBar: AppBar(
        title: Text("Awesome Quotes"),
        backgroundColor: Colors.redAccent,
        centerTitle: true,
      ),
      body: Column(
        children: quotes.map((quote) {
            return Text(quote.text + " - " + quote.author);
        }).toList()
      )
    );
  }
}
```

Now we can cycle through the list by using a button. Let's define a method as below:

```dart
void updateQuote() {
  setState(() {
    quotes.add(Quote(text: "Stay Hungry, Stay Foolish", author: "Steve Jobs"));
  });
}
```

Now we can use this method to update the state as below:

```dart
body: Column(
  children: quotes.map((quote) {
      return Text(quote.text + " - " + quote.author);
  }).toList()
)
```
