# Using Custom Fonts:

Download and place the fonts inside the `fonts` folder.

To use custom fonts, font must be first defined in `pubspec.yaml` file.

Configure custom fonts on `pubspec.yml`

```yml
  fonts:
    - family: IndieFlower
      fonts:
        - asset: fonts/IndieFlower-Regular.ttf
```

Now we can use the font in our application:

```dart
Text(
  'Hello World',
  style: TextStyle(
    fontFamily: 'IndieFlower',
  ),
)
```
