# Dart Primer

- [Dart Docs](https://dart.dev/language)

Note that dart is static type language but it can infer the type of the variable based on the value assigned to it. 

```dart
void main() {
  print("Hello world");
}

// to run the file
// dart <file>.dart
```


## 1. Data Types:

**i. Numbers:**

```dart
void main() {
  num x = 10; // can be int or double
  num y = 10.5;
  num z = -10.5;
  print(x);
  print(y);
  print(z);
}
```


**ii. Strings:**

```dart
void main() {
  String name = "asdasd";
  print(name);
}
```

Interpolation:

```dart

void main(){
    String name = "Surya";
    print("Hello $name");
}

```

**iii. Boolean:**

```dart
void main() {
  bool isTrue = true;
  bool isFalse = false;

  print(isTrue);
  print(isFalse);
}
```

**iv. Lists:**

```dart
void main() {
  List<String> names = ["asdasd", "asdasd"];
  print(names);
}
```

**v. Maps:**

```dart

void main(){
 // here, key is a string and value is number
 Map<String, num> ages = {"Surya": 22, "Ram": 20};
  print(ages);
}

```

**vi. Dynamic Type**

```dart
void main(){
    dynamic foo = "Hello";
    foo = 123;
    print(foo);
}
```


## 2. Functions:

Dart cannot run a file without a main function.

```dart
void main(){
    print("Hello world");
}
```

Here, ``void`` indicates that the function doesnt returns anything.

```dart

// returns  string, expects string
String greet(String name) {
  return "Hello, $name";
}

void main() {
  String foo = greet("Surya");
  print(foo);
}
```

Dart also supports use of arrow functions (for single-expression functions):

```dart
String greet(String name) => "Hello $name";

void main(){
    String name = "Surya";

    dynamic foo = greet(name);

    print(foo);
}
```
