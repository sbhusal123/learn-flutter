# Stateful Widget:

- Define a state class that extends from ``State<WidgetName>``.

- ``createState()`` method returns a ``_TestState`` object.

- The ``_TestState`` object is the state of the widget.

- Use ``setState(() { // body } )`` to set an state.

**Set State Signature:**

```dart
void setState(VoidCallback fn)

// setState(() { _myState = newValue; });
```

In the above signature, 
- ``VoidCallback`` is a callback function that takes no arguments and returns nothing.
- ``fn`` is the callback function that will be called when the state is set.


## Snippet:

```dart

// widget
class Test extends StatefulWidget {
  // returns a state
  @override
  State<StatefulWidget> createState() {
    return _TestState();
  }
}

// state
class _TestState extends State<Test> {
  
  // define state here:
  int foo = 0;
  int bar = 1;
  var baz = {"asd": 1, "asdasd": 2};

  @override
  Widget build(BuildContext context) {
    return Container();
  }

  // set state example
  void updateState() {
    setState(() {
      foo = foo+1;
      bar = bar+2;
      baz["asdasd"] = 3;
    });
  }
}
```

In the above snippet, 
- Test is the StatefulWidget.

It has a ``createState()`` method that returns a ``_TestState`` object.


- The ``_TestState`` object is the state of the widget.

It has a ``build()`` method that returns a ``Widget``.


## Example Usage:

In the example below, we have a state variable ``ninjaLevel``. Which is increamented each time on a click
of a floating Action button.

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(
    MaterialApp(
      home: NinjaCard(),
    )
  );
}


// Widget
class NinjaCard extends StatefulWidget {

  @override
  State<StatefulWidget> createState() {
    return _NinjaCardState();
  }
}


// State
class _NinjaCardState extends State<NinjaCard> {

  // defining state
  int ninjaLevel = 0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      backgroundColor: Colors.grey[900],
      appBar: AppBar(
        ....
        ...
      ),
      body: Padding(
        ....
        child: Column(
          ....
          children: [
            Text(
              ' Ninja Level: $ninjaLevel',
              style: TextStyle(color: Colors.amberAccent, fontSize: 28, fontWeight: FontWeight.bold)
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () {
          // setState triggers the build() method to be called again
          // Thus updating the UI with the new state
          setState(() {
            ninjaLevel = ninjaLevel+1;
          });
        },
        child: Icon(Icons.add),
      ),
    );
  }
}
```


State is updated with: using ``setState`` method.

```dart
onPressed: () {
    setState(() =>  ninjaLevel = ninjaLevel+1);
},

// onPressed: () {
//    setState(() =>  ninjaLevel = ninjaLevel+1);
// },
```
